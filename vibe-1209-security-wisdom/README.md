# 정보보안 명언 생성 웹앱

정보보안과 관련된 명언을 랜덤으로 표시하는 간단한 웹 애플리케이션입니다.

## 프로젝트 소개

이 프로젝트는 순수 HTML, CSS, JavaScript만을 사용하여 제작된 입문자 친화적인 웹앱입니다. 정보보안 전문가들의 지혜로운 명언을 랜덤으로 표시하여 보안에 대한 인식을 높이는 것을 목표로 합니다.

## 주요 기능

- **랜덤 명언 표시**: 30개의 정보보안 관련 명언 중 하나를 랜덤으로 표시
- **새 명언 보기**: 버튼 클릭으로 새로운 명언 생성
- **반응형 디자인**: 모바일, 태블릿, 데스크톱 모든 기기에서 최적화된 화면 제공
- **부드러운 애니메이션**: 명언 변경 시 페이드인 효과

## 파일 구조

```
vibe-1209-security-wisdom/
├── index.html      # 메인 HTML 구조
├── style.css       # 스타일링
├── script.js       # 명언 랜덤 표시 로직
└── README.md       # 프로젝트 설명서
```

## 사용 방법

1. 프로젝트 폴더를 열어주세요
2. `index.html` 파일을 웹 브라우저로 열어주세요
3. 페이지가 로드되면 자동으로 명언이 표시됩니다
4. "새 명언 보기" 버튼을 클릭하면 새로운 명언을 볼 수 있습니다

## 기술 스택

- **HTML5**: 시맨틱 마크업
- **CSS3**: 모던한 스타일링 및 애니메이션
- **JavaScript (Vanilla)**: DOM 조작 및 랜덤 로직

## 코드 설명

### HTML 구조
- `header`: 제목과 부제목
- `main`: 명언 카드와 버튼
- `footer`: 하단 메시지

### CSS 주요 기능
- 그라데이션 배경
- 카드 스타일 디자인
- 호버 효과
- 반응형 미디어 쿼리

### JavaScript 로직
- `securityQuotes`: 명언 배열
- `getRandomQuote()`: 랜덤 명언 선택 함수
- `displayQuote()`: 명언 화면 표시 함수
- 이벤트 리스너: 페이지 로드 및 버튼 클릭 처리

## 향후 개선 가능한 기능
